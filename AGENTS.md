# AGENTS.md

Instructions for AI agents working in **HomeProjects**.

## Edit policy (required)

**[docs/agent-guide.md](docs/agent-guide.md)** — especially:

1. **Read** all live docs in scope (`needs/`, `architecture/`, `portfolio/`, `projects/`, `docs/domain/`)
2. **Grep** the live tree for stale references when one area changes
3. **Archive** superseded content → `archive/YYYY-MM-DD-vN-name/` (archive may grow)
4. **Write live docs from scratch** — prefer full rewrites over small contradictory patches
5. **Root must stay consistent** for a first-time external reader
