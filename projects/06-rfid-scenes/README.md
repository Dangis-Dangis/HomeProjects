## Recommended

**Phone NFC first → optional ESPHome PN532 pads**

| | |
|---|---|
| **Cost** | $0–60 |
| **Setup** | 1–4 hrs |
| **Maintenance** | Low |
| **Feasibility** | ★★★★★ |
| **Scalability** | ★★★★ |

HA Companion NFC tags → `scene.turn_on` / cast macros. Add ~$12/reader only where a fixed tap point helps.

**Not for door locks** — convenience scenes only.
